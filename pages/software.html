List of GPL-Licensed Software for Game Development.

by Sean Stafford

The criteria for this list is not quite as specific as the game list and somewhat objective.
If you know of software not on the list that fits the criteria, please email me the name and/or link.
Here is the criteria:

  • The software must be licensed under the GPLv2, GPLv3, or AGPLv3.
  • The software's main purpose must be a task common in game development.

Here is the current list:

  • blender (GPLv3)
  • GIMP (GPLv2)
  • Inkscape (GPLv2)
  • Krita (GPLv2)
  • LMMS (GPLv2)
  • MyPaint (GPLv2)
  • Pencil2D (GPLv2)
  • pixeditor (GPLv3)
  • Tiled (GPLv2)
  • WorldForge (GPLv2)
